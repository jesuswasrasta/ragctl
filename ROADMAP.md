# Roadmap

## Q4 2025 - Stability & Core Features (Current)
- [x] Basic PDF/Image processing
- [x] OCR integration (EasyOCR)
- [x] Qdrant integration
- [ ] Improved error handling for corrupted PDFs
- [ ] Windows support verification

## Q1 2026 - Advanced Processing
- [ ] **Table Extraction**: Better handling of tables in PDFs
- [ ] **Multi-modal Support**: Extracting images from PDFs
- [ ] **Custom OCR Models**: Support for user-provided models
- [ ] **Parallel Processing**: True multi-core batch processing

## Q2 2026 - Ecosystem & Integrations
- [ ] **More Vector Stores**: Chroma, Pinecone, Weaviate
- [ ] **API Server**: REST API wrapper around the CLI
- [ ] **Plugin System**: Allow community plugins for custom loaders

## Long Term Vision
To become the standard "ETL for RAG" tool - handling the messy reality of unstructured data so developers can focus on their LLM applications.
